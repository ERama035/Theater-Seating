This Java project has been compiled against Java 1.8 but should be able to compile and run against older versions (up to 1.5)

I apologise, if code formatting is messed up at your end. I have line wrap at 170, tab is space-only and tab size is 4. 

--------------------
How to run.
--------------------

TheatreSeating class has "main" method.
After successful compile, run it. It should prompt you to enter 'Theater Layout and ticket requests'. It should be in format given into original problem.

Prompt: Please enter Theater Layout and Ticket requests and then enter 'done'.

Sample input

6 6
3 5 5 3
4 6 6 4
2 8 8 2
6 6

Smith 2
Jones 5
Davis 6
Wilson 100
Johnson 3
Williams 4
Brown 8
Miller 12
done (Press enter)

Output

Ticket distribution

Smith Row 1 Section 1
Jones Row 2 Section 2
Davis Row 1 Section 2
Wilson Sorry, we can't handle your party.
Johnson Row 2 Section 1
Williams Row 1 Section 1
Brown Row 4  Section 2
Miller Call to split party.
